Set Dawn of Ashes First Edition
Effect Banish Horn of Beastcalling: The next Beast ally card you activate this turn costs 3 less to activate. Class Bonus: Draw a card. (Apply the additional effect only if your champion's class matches this card's class.)
Rules 2022-12-27 – The cost decrease only affects Beast ally cards that are activated, not materialized.2022-12-27 – The type of cost decreased is based on the card's default cost type. In most cases, it will decrease reserve cost because it only affects activation costs. However, if a card can be activated from the material deck and has a memory cost as its default cost, the memory cost will decrease by 3.
