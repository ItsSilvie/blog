This card is part of the Dawn of Ashes First Edition set.

Effect Banish Channeling Stone: The next card you activate this turn costs 2 less to activate.
Rules 2022-12-27 – The cost decrease only affects cards that are activated, not materialized.2022-12-27 – The type of cost decreased is based on the card's default cost type. In most cases, it will decrease reserve cost because it only affects activation costs. However, if a card can be activated from the material deck and has a memory cost as its default cost, the memory cost will decrease by 2.
